# 期权分析项目

本项目用于期权风险指标分析和投资研究。

## 文件说明

- `投研.ipynb`: Jupyter 笔记本，包含期权分析的主要逻辑和可视化
- `OptionRiskMetrix.py`: 期权风险指标计算相关函数
- `ScenrioApp.py`: 场景应用程序
- `commodity_common_functions.py`: 商品相关的公共函数库

## 使用方法

1. 安装依赖包

   ```
   pip install pandas numpy matplotlib openpyxl
   ```

2. 运行投研笔记本
   ```
   jupyter notebook 投研.ipynb
   ```

## 注意事项

确保已安装依赖库，特别是`openpyxl`库用于 Excel 文件读写。
